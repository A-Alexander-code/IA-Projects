# Gender Recognition Model

![image](https://github.com/A-Alexander-code/IA-Projects/blob/main/z_image/AI-gender-eng.jpg)

This project focuses on developing a gender recognition model using deep learning techniques. The model is designed to classify gender based on visual data, utilizing convolutional neural networks (CNNs) for accurate prediction.
Project Overview
Introduction

The goal of this project is to create a robust and accurate gender recognition model. The model is trained on a dataset of labeled images and is capable of distinguishing between different gender classes. Through the use of advanced machine learning techniques and rigorous evaluation, this project aims to provide a reliable solution for gender classification.
Key Features

- Model Architecture: EfficientNet-B0, a state-of-the-art CNN architecture, was used for training, balancing accuracy and efficiency.
- Performance Metrics: The model's performance was evaluated using precision, recall, F1 score, and Matthews Correlation Coefficient (MCC).
- Visualization: Confusion matrices, ROC curves, and training curves were used to analyze the model's performance.
- Model Saving and Loading: The trained model was saved for future use and can be easily loaded for deployment.

## Performance Metrics

### The model achieved the following metrics on the test dataset:

- Precision: 0.9412
- Recall: 0.9389
- F1 Score: 0.9386
- Matthews Correlation Coefficient: 0.8790

## Getting Started
### Prerequisites

* Python 3.8 or higher
* PyTorch
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Conclusion

This gender recognition model has shown strong performance on the provided dataset and offers a solid foundation for further development. It can be deployed in various applications, from security systems to user experience enhancements.
